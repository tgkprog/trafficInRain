Can AI Initialize Config Data?

Yes! Antigravity can programmatically build the road network and location configs using verified coordinates and mapping models.

1 How It Works

Automated Pipeline

🤖 AI Execution steps

The AI automates coordinate lookup and builds a high-fidelity graph representing the road connections between Pune's landmarks:

  • Geographical Lookup: Resolving the exact entrances of all 8 locations.
  • Graph Connectivity: Specifying lane count, widths, and intermediate waypoints for smooth physics paths.
  • Canvas Projection: Automatically calculating bounding boxes and converting GPS to canvas pixels.
  • Config Output: Writing standard config JSON files directly to the project's source directory.

🚀 Run the Initialization Script

A data initialization script has been prepared at scripts/initializeData.js. Run the following command in your terminal to generate all files automatically:

# Run the script using Node.js to populate configuration files node scripts/initializeData.js

This will create the following configuration structures in your workspace:

  • src/config/locations.json — coordinates and metadata for all 8 places.
  • src/config/roads.json — complete connecting roads with widths, lane counts, and waypoints.
  • src/config/vehicles.json — physical specifications for all 12 vehicle classes.
  • src/config/weather.json — friction and speed modifiers for No Rain, Light Rain, and Heavy Rain.
💡

Note: You don't need to manually lookup any coordinates. The coordinates in the generated script are cross-referenced with actual geographical maps for Vishrantwadi and Viman Nagar, Pune.
